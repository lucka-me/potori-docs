# 管理数据
Potori会在本地保存数据，您可以在偏好设置中管理数据。

## 提名
Potori会将提名保存在本地数据库中，您可以将它们导出至JSON文件，从JSON文件中导入以及清空所有提名。

### 导入Wayfarer JSON<Badge type="tip" text="Web" vertical="top"/>
Potori可以导入[Wayfarer API](https://wayfarer.nianticlabs.com/api/v1/vault/manage "Wayfarer API")中的数据以更新提名的名称和坐标。

### Google Drive同步
您可以通过Google Drive在各个设备和平台间同步提名数据。数据将被存储在只有Potori能访问的私有文件夹中。

# Brainstorming<Badge type="tip" text="Web" vertical="top"/>
网页应用提供了集成了Brainstorming功能，它能够从线上数据库中查询记录并保存在本地数据库。您也可以将它们导出至JSON文件或从JSON文件中导入。
